Projet : Site E_Cmmerce Azar Cosmétique


Membres du groupe :

OULTACHE Ines
NAILI Ines
CISSE Issa


description du projet :

Ce projet est un site web de e-commerce fictif spécialisé dans les produits 
cosmétiques naturels. L'objectif était de créer une interface fonctionnelle 
permettant aux utilisateurs de naviguer dans le catalogue, de découvrir les 
produits et de se connecter à leur espace personnel.

Fonctionnalités principales :

   Interface utilisateur : Un design clair et responsive qui s'adapte aux différents écrans,
 avec une identité visuelle cohérente.

   Système de connexion : Un formulaire de login sécurisé côté client qui vérifie les identifiants 
 via le stockage local du navigateur (localStorage).

   Navigation : Une structure de menu simple pour passer facilement entre la page d'accueil, la 
page des produits et l'espace de connexion.

Outils utilisés :

HTML5 : Pour structurer le contenu de manière sémantique.

CSS3 : Pour la mise en forme, la gestion de la mise en page (Flexbox/Grid) et les effets visuels.

JavaScript : Pour dynamiser la page, gérer la validation des formulaires et simuler la persistance des données.
